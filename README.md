# open_cv

## Setup

- Clone (or download) the repository:
```ShellSession
git clone git@github.com:Paulinakhew/open_cv.git
```

- Download all the necessary packages:

* **MacOS Users**
```ShellSession
$ pip3 install -r requirements.txt
```

* **Linux Users**
```ShellSession
$ pip install -r requirements.txt
```
